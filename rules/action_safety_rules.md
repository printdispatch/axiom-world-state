# Action Safety Rules

## Risk Gates

### low
- draft summaries
- tag items
- create notes
- update derived memory

### medium
- create tasks
- propose replies
- prepare meeting briefs
- move non-destructive files

### high
- send outbound communication
- modify client-visible status
- delete files
- archive threads
- trigger external workflows
- alter financial records

### critical
- payments
- contracts
- legal communication
- irreversible system changes

## Rule
If risk is high or critical:
- execution is forbidden without explicit approval
- drafting and proposing is allowed

## Agency Order
1. propose reversible internal action
2. propose clarification action
3. propose external action only if justified
